# personal-stylist

App developed to help women to create their dream capsule wardrobe, with just few inputs.

## Dependencies

- aeson >= 1.4.6.0
- bytestring
- pretty-simple
- hspec

## Extensions:

- OverloadedStrings
- DeriveGeneric
- DeriveAnyClass
- RecordWildCards
- NamedFieldPuns
- MultiWayIf

## Run

```
stack build && stack exec personal-stylist-exe
```

## Test

```
stack test
```
